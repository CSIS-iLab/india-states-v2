Arunachal Pradesh issues draft APSERC (Arunachal Pradesh State Electricity Regulatory Commission) State Grid Code Regulations, 2025. These regulations repeal earlier 2018 regulations and set out the rules, guidelines, and standards to be followed by all users of the State Transmission System. It aims to improve grid stability, standardize operational practices, and facilitate clear communication and data sharing between utilities.
